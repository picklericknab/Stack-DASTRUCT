What is a Stack?

A Stack is a linear data structure that follows the Last In, First Out (LIFO) principle — the last element inserted is the first one to be removed.

Real-world Examples

  • 🍽️ Stack of plates
  • ↩️ Browser back button
  • ✏️ Undo / Redo in editors
  • 📞 Function call stack
